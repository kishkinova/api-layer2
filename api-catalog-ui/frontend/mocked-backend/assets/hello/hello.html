Welcome to mocked backend

These are the available endpoints

  • /apicatalog/api/v1/auth/login - Enables you to log in. Use user as username and password. The login process takes 2 seconds to simulate slow network.
  • /apicatalog/api/v1/auth/logout - Logs out the currently logged user.
  • /apicatalog/api/v1/containers - Serves a static json (`mocked-backend/assets/containers`).
  • /apicatalog/api/v1/containers/some-down - Simple get endpoint that toggles the status of some tiles.
  • /apicatalog/api/v1/containers/:serviceID - Serves the same information as /apicatalog/api/v1/containers but for a specific serviceID.
  • /apicatalog/api/v1/apidoc/:serviceID/:version - Serves apidoc for a specific serviceID and version.
The endpoints above reflect the real services with the /apicatalog/api/v1/containers/some-down being an exception.
